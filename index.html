Нужна настройка Firebase

Чтобы данные были общими для всех, нужно подключить Firebase. Это бесплатно и займёт ~5 минут.

  1. Зайди на console.firebase.google.com → создай проект
  2. В проекте: Build → Realtime Database → Create database (выбери "test mode")
  3. В настройках проекта (⚙️) → General → Your apps → Web → скопируй конфиг
  4. Открой файл index.html и замени значения в блоке FIREBASE CONFIG в начале скрипта
  5. Там же установи свой пароль администратора в ADMIN_PASSWORD

⚠️ В Firebase Console → Realtime Database → Rules установи правила из README ниже для защиты данных.
